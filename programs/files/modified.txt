The Silt Programming Language
=============================

Silt is a statically-typed, expression-based language.
It uses immutable bindings exclusively.
Pattern matching is the primary branching construct.
Match expressions replace if/else entirely.

Functions are defined with the fn keyword.
The pipe operator passes values through chains.
Trailing closures improve readability.
Lists, tuples, and maps are built-in types.
Records provide named fields.

Error handling uses Result and Option types.
The standard library is deliberately small.
The loop expression provides safe iteration.
Documentation lives in the docs folder.
